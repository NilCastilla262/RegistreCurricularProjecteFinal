Eines per a Administradors

Com a administrador, tens accés complet a totes les funcions de gestió educativa:

  • Crear SDA: Dissenyar noves activitats d'aprenentatge pels alumnes.
  • Veure SDA: Consultar i modificar les activitats existents.
  • Veure Resum: Obtenir informes de progrés per cada grup.
